Naive Bayesian

I am going to start out with Naive Bayesian as to give me an insight into the features and what is important before running more in- depth models. I will do cross validation for better assurance and make it with 10 folds.

# Manually calculate precision and recall
calc_precision <- function(predictions, actuals) {
  tp <- sum(predictions == actuals & actuals == levels(actuals)[2])
  fp <- sum(predictions != actuals & actuals == levels(actuals)[1])
  if (tp + fp == 0) return(0)  # Avoid division by zero
  return(tp / (tp + fp))
}

calc_recall <- function(predictions, actuals) {
  tp <- sum(predictions == actuals & actuals == levels(actuals)[2])
  fn <- sum(predictions != actuals & actuals == levels(actuals)[2])
  if (tp + fn == 0) return(0)  # Avoid division by zero
  return(tp / (tp + fn))
}

calc_f1_score <- function(predictions, actuals) {
  precision <- calc_precision(predictions, actuals)
  recall <- calc_recall(predictions, actuals)
  if (precision + recall == 0) return(0)  # Avoid division by zero
  return(2 * precision * recall / (precision + recall))
}

cv_function <- function(df, target_index, nFolds, seedVal, classification_func, metrics_list) {
  set.seed(seedVal)
  folds <- createFolds(df[[target_index]], k = nFolds, list = TRUE)

  cv_results <- lapply(folds, function(fold_indices) {
    train_indices <- setdiff(seq_len(nrow(df)), fold_indices)
    test_indices <- fold_indices
    
    train <- df[train_indices, -target_index, drop = FALSE]
    test <- df[test_indices, -target_index, drop = FALSE]
    
    train_target <- df[train_indices, target_index]
    test_target <- df[test_indices, target_index]

    model <- classification_func(x = train, y = train_target)
    predictions <- predict(model, test)

    metrics_data <- list()
    if ("ACC" %in% metrics_list) {
      acc <- mean(predictions == test_target)
      names(acc) <- "ACC"
      metrics_data[["ACC"]] <- acc
    }
    if ("PRECISION" %in% metrics_list) {
      precision <- calc_precision(predictions, test_target)
      names(precision) <- "PRECISION"
      metrics_data[["PRECISION"]] <- precision
    }
    if ("TPR" %in% metrics_list) {
      tpr <- calc_recall(predictions, test_target)
      names(tpr) <- "TPR"
      metrics_data[["TPR"]] <- tpr
    }
    if ("F1" %in% metrics_list) {
      f1 <- calc_f1_score(predictions, test_target)
      names(f1) <- "F1"
      metrics_data[["F1"]] <- f1
    }
    
    return(metrics_data)
  })

  # Combine fold-specific metrics into a dataframe
  cv_results_df <- do.call(rbind, cv_results)
  
  return(cv_results_df)
}

Feature Importance

feature_importance <- function(model) {
    feature_stats <- lapply(names(model$tables), function(feature) {
        # Extract the stats and ensure it's a data table
        stats <- as.data.table(model$tables[[feature]])

        # Ensure that values are numeric
        for (col in names(stats)) {
            if (!is.numeric(stats[[col]])) {
                stats[[col]] <- as.numeric(stats[[col]])
            }
        }

        stats[, feature := feature]  # Adding feature names as a column
        melted_stats <- melt(stats, id.vars = "feature", variable.name = "Class", value.name = "Probability")
        return(melted_stats)
    })

    # Combine all feature stats
    feature_stats <- do.call(rbind, feature_stats)
    # Calculate mean probabilities grouped by feature and class
    importance_data <- feature_stats[, .(MeanProbability = mean(Probability, na.rm = TRUE)), by = .(feature, Class)]
    return(importance_data)
}

##Interpret what mistakes in predictions would mean to marketing financial products to individuals. ###If some with low income was categorized as having high income and a high income product was marketed to them what are the ramifications of this if they are unable to repay the loan? Misclassifying a low-income individual as having a high income and marketing high-income financial products to them can lead to several significant issues, both for the individual and for the company offering the products:

  1. Financial Strain on the Individual: If a person with a low income is incorrectly targeted with products meant for high-income individuals, they may face financial products (like loans or credit cards) with terms that are not suitable for their income level. This could lead to high monthly payments that they cannot afford, increasing the risk of default.
  2. Increased Default Rates: Marketing inappropriate financial products to individuals who cannot afford them could lead to higher default rates. This can happen if the individuals are enticed by the perceived value or status associated with high-income products and overextend themselves financially.
  3. Credit Score Impact: For the individuals, defaulting on loans or credit payments can severely damage their credit scores. A poor credit score can affect their ability to secure loans in the future, obtain favorable interest rates, or even affect renting and employment opportunities. 4)Reputational Damage: From a company’s perspective, if it becomes known that they routinely market inappropriate products to unsuitable candidates, it can lead to reputational damage. This might result in a loss of consumer trust and could deter potential customers.
  4. Operational Costs: Dealing with the fallout of bad debt, increased default rates, and regulatory compliance issues can lead to increased operational costs. This includes costs related to debt collection, legal fees, and potential settlements or fines.

What if a high income individual has a less profitable financial product marketed to them because we accidentally predict them to have low income?

If a high-income individual is mistakenly classified as having low income and is marketed less profitable financial products, the ramifications include missed opportunities for both the individual and the financial institution. The individual may miss out on more advantageous investment opportunities or favorable terms on products that match their financial capabilities, potentially limiting their financial growth. Simultaneously, the financial institution loses out on higher revenue from premium services that could have been offered, and it may negatively impact customer satisfaction and trust, as the client might feel undervalued or poorly assessed, leading them to seek more personalized services from competing firms.
